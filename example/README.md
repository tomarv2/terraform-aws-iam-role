terraform-aws-iam-role
======================

Terraform module to create AWS IAM Role. 

1. Role for IAM User
2. Role for External AWS Account
3. Role for Instance Profile
4. Role for Lambda


Usage
-----
This module will create an IAM Role and its Trust Relationships policy document.
